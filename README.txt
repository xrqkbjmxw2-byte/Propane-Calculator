How to run locally (Mac):
1) Open Terminal in this folder.
2) Run:  python3 -m http.server 5500
3) Visit: http://localhost:5500  (on iPhone: use your Mac's LAN IP, e.g., http://192.168.1.50:5500)
4) Add to Home Screen to install.

Quick deploy (any host):
- Upload all three files to a static host (GitHub Pages, Netlify, etc.).
- Open the URL on your phone; you should see the install prompt.
